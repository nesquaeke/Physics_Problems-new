8. WavesConcept: To be a proper traveling wave, a function MUST take the specific grouped shape of $(x - vt)$ or $(x + vt)$. If $x$ and $t$ are not locked together in this exact linear way, the wave will distort as it moves, so it doesn't solve the wave equation.Step-by-Step Solution:a) $y(x,t) = A \cos(kx^2 - \omega t)$No. It has an $x^2$. As $x$ gets bigger, the wave gets squished. It fails the wave equation derivative test.b) $y(x,t) = A(x-vt)^2$Yes. Even though it's squared outside, the inside is exactly $(x-vt)$. If you take the second derivatives, it perfectly satisfies the wave equation.c) $y(x,t) = A \log(x+vt)$Yes. The inside is exactly $(x+vt)$, which represents a wave moving to the left without changing its basic shape.
