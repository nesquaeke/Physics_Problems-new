3. Superposition Principle
Concept: "Superposition" is a fancy word for "adding waves together." If two waves crash into each other, you just add their heights. We have two waves moving in opposite directions (one has $- \omega t$, the other has $+ \omega t$).Step-by-Step Solution:The Equation: We add $y_1$ and $y_2$:$$y(x,t) = A \sin(kx - \omega t) + A \sin(kx + \omega t)$$In trigonometry, there is a shortcut formula for adding sines: $\sin(a) + \sin(b) = 2 \sin(\frac{a+b}{2}) \cos(\frac{a-b}{2})$. Using this, we get the standing wave equation:$$y(x,t) = 2A \sin(kx) \cos(\omega t)$$Finding Nodes: Nodes are the "dead spots" on a string that never move. This happens when the sine part is equal to zero: $\sin(kx) = 0$.Sine is zero at $0, \pi, 2\pi, 3\pi...$ which we write as $n\pi$ (where $n$ is $0, 1, 2...$).So, $kx = n\pi$.We know $k = \frac{2\pi}{\lambda}$. Let's plug it in:$$\frac{2\pi}{\lambda} x = n\pi$$Cancel out the $\pi$ and solve for $x$:$$x = \frac{n \lambda}{2}$$Nodes appear every half-wavelength ($0, \lambda/2, \lambda, 1.5\lambda...$).
