7. Standing Wave ModesConcept: "Antinodes" are the biggest bumps on a standing wave. The number of antinodes equals the harmonic number ($n$). Each bump is exactly half of a wavelength ($\lambda/2$).Step-by-Step Solution:We have 4 bumps (antinodes), so the string holds four "half-wavelengths".$$L = 4 \cdot \left(\frac{\lambda}{2}\right) = 2\lambda$$The length of the string $L = 80$ cm ($0.8$ m).$$0.8 = 2\lambda$$Divide by 2:$$\lambda = 0.4 \text{ meters (or 40 cm)}$$
