## 2) Range Optimization

For projectile motion, the range is:

`R(theta) = (v0^2 * sin(2theta)) / g`

We want to find the angle that gives the maximum range.

### Step 1: Identify what changes

For a fixed initial speed `v0` and constant gravity `g`, the only changing part is:

`sin(2theta)`

So the range is maximum when `sin(2theta)` is maximum.

### Step 2: Maximum value of sine

The largest possible value of sine is:

`1`

So we need:

`sin(2theta) = 1`

This happens when:

`2theta = 90 degrees`

So:

`theta = 45 degrees`

**Final answer:**  
The maximum range is achieved at:

`45 degrees`

### Extra interpretation

This means the projectile goes farthest when the launch angle is exactly halfway between horizontal and vertical.
